Date Completed

What It Is
A Undertale-inspired roleplaying game that is different each time based on how the player answers pregame questions. The game discusses issues of class as it relates to education. It held a 4.8/5.0-star rating at a community game jam
Technologies Used
HTML, CSS, Bootstrap, JavaScript, jQuery, and Photoshop
Plans for Future Improvement
  • Optimize for difference screen sizes. I underestimated the amount of effort the canvas element would require of me, so I hardcoded a lot of coordinates in, which causes it to break a bit if played with certain monitors
  • Port the game to Unity (maybe)
  • Add more levels
  • Modularize the code
