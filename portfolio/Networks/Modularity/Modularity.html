What It Is
A parser for the AMPL object language. It goes through a file in the metalanguage, SML (NJ), and determines whether the source file contains legal expressions and declarations. This doesn't include type checking (that's the type engine's job)
Technologies Used
SML (NJ), ML Lex, and BASH
